public subroutine readPositionFiles(neutralPositionsFile, chargedPositionsFile, atomNames, atomMasses, nAtoms, atomD, atomM)

Read the positions for the neutral and charged cells to calculate the base displacements with a charge. Also read in the number of atoms and the atom masses

Note

This subroutine takes in positions in XSF format. To get this, you will need to first relax the positions for the neutral and charged cells, then run ${QE-5.3.0_Path}/PW/tools/pwo2xsf.sh -lc [relaxation output file] > [coordinate file name] on each of the output files. What you use in place of [coordinate file name] will need to be included in the input file.
